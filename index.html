Plotly

The purpose of this project is to use the Plotly library in JavaScript in combination with HTML and CSS to build an interactive dashboard of real belly button microbe data conducted by North Carolina State University. I used JSON to read the microbe sample data and Plotly in JS to create an interactive bar chart and bubble chart complete with tooltips. Users can use the dropdown menu to select IDs to view their different levels of microbial species.



D3

This project uses the JavaScript library D3 to compare the health risks facing particular demographics provided by the US Census Bureau and the Behavioral Risk Factor Surveillance System. Three interactive scatter plots compare poverty levels, median age, and household income on one axis with lack of healthcare, smokers and obesity levels on another axis from all 50 states across the country. Users can click and on each label of the axis to see how variables from one compare to the other. Users can also hover over each point to see more information.



Web Mapping

The purpose of this project is to use the Leaflet library in JavaScript to build an interactive map visualizing earthquake data provided by the United States Geological Survey. Earthquakes that occurred in the last 7 days are plotted on the map based on latitude and longitude. Data markers reflect the magnitude of the earthquake by size and depth according to color. Earthquakes with higher magnitudes appear larger and earthquakes with greater depth appear darker in color. The map includes popups that provide additional information about the earthquake when a marker is clicked. Users can zoom, move and toggle between 3 different types of maps and select or deselect different map layers.



Tableau

This project uses Tableau to analyze data from the New York Citi Bike Program. Dashboards were created to show a recap of basic data from December 2019 looking usage by gender and subscriber type. Users can click on charts and maps to show more information.


*Users must have a public or paid subscription of Tableau to view this project.
